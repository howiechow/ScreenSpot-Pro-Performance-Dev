# ScreenSpot-Pro Performance Dev

This repository collects the performance of different models on the ScreenSpot-Pro GUI grounding benchmark.

## ScreenSpot-Pro Leaderboard Results

- UGround-V1-7B, 31.1%, https://github.com/OSU-NLP-Group/UGround
- Aguvis-7B, 22.9%, https://github.com/xlang-ai/aguvis
- AriaUI (MOE, 3.9B active), 11.3%, https://ariaui.github.io/

## About ScreenSpot-Pro

ScreenSpot-Pro is a GUI grounding benchmark for professional high-resolution computer use. The dataset can be found on HuggingFace at: https://huggingface.co/datasets/likaixin/ScreenSpot-Pro

The official leaderboard is available at: https://gui-agent.github.io/grounding-leaderboard/
