# ScreenSpot-Pro Performance Development

This repository collects and tracks the performance of different models on the ScreenSpot-Pro GUI grounding benchmark.

## ScreenSpot-Pro Benchmark

ScreenSpot-Pro is a GUI grounding benchmark for professional high-resolution computer use. It tests models on various professional applications across Development, Creative, CAD, Scientific, Office, and Operating System categories.

- **Dataset**: [likaixin/ScreenSpot-Pro on HuggingFace](https://huggingface.co/datasets/likaixin/ScreenSpot-Pro)
- **Leaderboard**: [ScreenSpot-Pro Leaderboard](https://gui-agent.github.io/grounding-leaderboard/)
